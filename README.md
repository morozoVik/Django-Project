# Skystore - Интернет-магазин цифровых товаров

## Описание проекта
Skystore - это интернет-магазин для продажи цифровых товаров: плагинов, примеров кода, телеграм ботов и других IT-решений.

## Функциональность
- Главная страница с каталогом товаров
- Страница контактов с формой обратной связи
- Адаптивный дизайн на Bootstrap

## Технологии
- Python 3.x
- Django 5.2
- Bootstrap 5.3
